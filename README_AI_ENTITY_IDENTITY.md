# StegID AI Entity Identity (v1)

This patch adds a **portable identity spec** for StegVerse AI entities.

## Contents
- `docs/AI_ENTITY_IDENTITY_SPEC_V1.md` — normative spec
- `schemas/ai_entity_identity_v1.json` — JSON schema for portable profiles
- `src/identity/entity_identity.py` — tiny helper utilities (optional)

This is additive and should not break existing receipt verification.
