## Reporting a Vulnerability

**Preferred (private disclosure):**
- Use **GitHub Security Advisories** for this repository.

**Non-sensitive issues:**
- Open a GitHub issue titled:
  `SECURITY: <short description>`

Please include:
- impact
- reproduction steps
- suggested mitigation

Do not publish exploit details until a fix is available.

## Scope

StegID Phase 1:
- stores no PII
- stores no biometrics
- stores no DNA
- focuses only on continuity of control

See `THREAT_MODEL.md` for full analysis.
